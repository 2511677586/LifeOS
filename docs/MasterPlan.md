# Master Plan

LifeOS is being developed in layered milestones so the capture experience can grow into a structured personal knowledge system without breaking the existing memory foundation.

## Milestone 1 — Memory Foundation
Milestone 1 established the initial capture workflow: multiline text input, saving, Markdown persistence, and a recent records browser.

## Milestone 2 — Knowledge Foundation
The second major system layer is Knowledge Foundation. It builds on the memory capture flow and introduces the structural basis for a durable knowledge system.

Knowledge Foundation provides:
- structured metadata
- knowledge classification
- tags
- indexing
- search
- relationships
- knowledge browsing

Milestone 2 remains focused on knowledge structure and retrieval. AI features are not part of this milestone.
